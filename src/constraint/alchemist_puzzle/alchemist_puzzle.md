# FORMULA IV-C — *Lock compound, wax-stoppered jar, third shelf*
*Reconstruct from this if the jar is lost. Done it enough times it's rote, but write it down.*

---

Four vessels. Glass, brass, copper, clay. They'll have residue — that's fine, it doesn't interfere.

**The vials:**
* The orange pours thin, nearly water. No smell. Clear enough to read through.
* The black has a sharp bite — set it out before you uncork it, and don't lean in.
* The purple is pungent; you'll smell it across the room. Lingers.
* The green moves slow. Sweet. Like warm honey.

**The sequence:**
1. First, the one you can see through — pour it and let it settle before the others cloud it.
2. Second, stand back when you open it. The second cup is not forgiving of curiosity.
3. Third — watch the lip of the cup. It will blush before you've finished pouring the fourth. The blue won't spoil the compound; ignore it and continue.
4. Last is always the green, and always the clay. Clay weeps if you leave the green in anything else.
5. Pour into the channel, the lock will open*
